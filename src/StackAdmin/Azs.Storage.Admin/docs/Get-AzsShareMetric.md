# Get-AzsShareMetric

## SYNOPSIS
Returns a list of metrics for a storage share.

## SYNTAX

```
Get-AzsShareMetric -FarmId <String> -ResourceGroupName <String> -ShareName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
Returns a list of metrics for a storage share.
